Le CSS pour décorer le Web

1. Un document en HTML /CSS pour mieux connaître le HTML / CSS

Là encore, vous allez utiliser ce document HTML / CSS pour compléter / conserver votre cours.

Pour suivre ce cours / activité, vous devez suivre à nouveau les consignes suivantes :

  • Copier ce fichier NSI_6_2_CSS.html dans un dossier de vos documents.
  • Ouvrir ce dossier à partir du logiciel Visual Studio Code (Fichier / Ouvrir le dossier).
  • Cliquer sur le fichier NSI_6_2_CSS.html pour le visualiser dans VSC.
  • Cliquer sur "Live server" en bas à droite de la barre bleue, tout en bas.

La suite de l'activité consistera à...

  1. compléter le contenu du cours à partir d'un autre site.
  2. utiliser ces nouvelles connaissances pour améliorer la mise en forme de cette page.

Vous aurez donc besoin de...

  • l'adresse du site web Mon lycée numérique, chapitre CSS.
  • et peut-être (re)lire le document PDF Memo1, qui est un résumé de cours sur le HTML / CSS
  • et peut-être (re)lire le document PDF Memo2, qui est mémo, concentrant les balises essentielles à connaître

Remarque : Dans l'ensemble de ce document, ne pas hésiter à effacer du texte si vous considérez qu'il ne vous sera pas utile pour réviser.

2. Le CSS, pour mettre en forme votre contenu

A partir d'ici, vous devez copier / coller / modifier les parties de cours intéressantes pour vous concernant le langage CSS.

Voici un rappel des consignes, complété :

  • Il faut respecter l'usage des balises (ex : balises de titres)
  • Il faut organiser correctement votre code HTML (indentation, pas de lignes trop longues, pas trop de sauts de lignes inutiles,...)
  • Il faut bien évidemment renuméroter vos chapitres
  • Ne conservez pas tous les chapitres mais uniquement ce qui vous intéresse. Par exemple, je ne pense pas utile de reprendre ces points :
    • Les références à jsfiddle.net
    • Tout ce qui a un rapport avec leur projet de classe (Chapitres 2 et 4).
  • A contrario, si vous avez le temps, il ne me semblerait pas inutile de compléter ces points par...
    • Les attributs les plus utiles, comme font-family, text-align, color, et bien encore...
    • Quelques notions de Webdesign.
  • Lorsque vous aurez fini, vous pourrez effacer ces consignes pour alléger votre page.

3. Maitenant que votre cours sur le CSS est prêt...

Dans un premier temps, et si vous ne l'avez pas déjà fait, aidez-vous du Mémo1 (chap. III.1) pour extraire le code CSS de cette page vers un nouveau fichier style.css. Attention, dans ce cas, il ne faut plus utiliser les balises <style>, ni dans le fichier html, ni dans le fichier css.

Remarque : Il est toujours préférable de séparer le code HTML du code CSS dans deux fichiers séparés car cela clarifie le code et améliore sa maintenabilité. Ce principe sera également à appliquer lorsque nous utiliserons d'autres langages, comme Javascript ou Brython.

Suivant le temps et l'envie que vous avez à consacrer à cette partie, je vous invite à améliorer le design de cette page.

Par exemple, en...

  • changeant la police de caractère des titres et / ou des paragraphes.
  • changeant certaines couleurs.
  • mettre certains mots importants en gras pour les rendre plus visibles.
  • ajoutant des documents illustratifs, comme des images, schémas ou photos.
  • ajoutant des liens hypertexte vers des sites qui vous ont aidé à créer ce cours.
  • ajoutant un liens hypertexte dans vos pages web (NSI_6_1 et NSI_6_2) pour naviguer facilement de l'une à l'autre.
