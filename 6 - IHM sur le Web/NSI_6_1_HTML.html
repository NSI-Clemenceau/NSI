a. HTML : Définition

L'HTML est un langage informatique utilisé sur l'internet. Ce langage est utilisé pour créer des pages web. L'acronyme signifie HyperText Markup Language, ce qui signifie en français "langage de balisage d'hypertexte". Cette signification porte bien son nom puisqu'effectivement ce langage permet de réaliser de l'hypertexte à base d'une structure de balisage.

Pour l'instant, nous allons retenir deux éléments de cette définition «conçu pour représenter les pages web» et «un langage de balisage».

Grâce au HTML vous allez pouvoir, dans votre navigateur (Firefox, Chrome, Opera,....), afficher du texte, afficher des images, proposer des hyperliens (liens vers d'autres pages web), afficher des formulaires et même maintenant afficher des vidéos (grâce à la dernière version du HTML, l'HTML5).

HTML n'est pas un langage de programmation (comme Python par exemple), ici, pas question de conditions, de boucles....c'est un langage de description.

En fait le HTML, permet de remplir un page web ou chaque élément est contenu dans une balise ( dans une capsule). Chacune des balises ayant des propriétés que l'on peut personnaliser à volonté.

4. Que retenir ?

A minima...

  • Le HTML est un langage de description, permettant de créer des pages web, en y ajoutant du contenu (texte, hypelien, image, vidéo,...)
  • Le HTML est un langage de balisage.
  • Le plus souvent, le contenu est compris entre une balise ouvrante <balise> et une balise fermante </balise>. Il existe toutefois certaines exceptions à cette règle (ex : <meta charset="utf-8">, <br/>,...).
  • Les balises "s'imbriquent" les unes aux autres pour encadrer le contenu.
  • Il faut connaître le rôle de la dizaine de balises utilisées dans ce document HTML, sans pour autant savoir les utiliser précisément. Par exemple, il faut comprendre que du texte contenu entre deux balises h1 va servir de titre principal.
  • L'indentation n'est pas obligatoire en HTML mais elle est très fortement conseillée.
  • On peut ajouter des attributs à certaines balises (ex : href=, alt=, src=,...).

Au mieux...

  • Il faut connaître la structure de base d'une page web minimale, avec ses balises essentielles.
  • Il faut savoir utiliser les balises communes, comme celles contenant un titre, un paragraphe, un texte en gras, un hyperlien, une image, une liste à puce,...
  • Dans certains cas (hyperlien, image,...), des attributs sont nécessaires au fonctionnement de la balise : href, alt, src,...
  • Quelques autres balises habituelles sont aussi bonnes à connaître, comme le saut de ligne <br/>, le tableau,...
