Comment rendre une page web interactive avec Javascript ?

Remarque : n'hésitez pas à appliquer votre fichier CSS à cette page HTML pour la rendre plus attrayante.

1. Construire un cours interactif

Cette fois encore, vous devrez construire votre cours à partir du site web Mon Lycée Numérique

Il vous faudra donc :

  • Lire attentivement cette page.
  • Afficher son code source (Crt+U) pour faciliter les "copier / coller" ultérieurs.
  • Copier / coller dans ce document HTML les éléments importants à retenir.
  • Faire des recherches complémentaires au besoin pour compléter votre cours.
  • Faire les exercices proposés, directement sur ce document de cours HTML.

Ce qu'il n'est pas indispensable de faire :

  • Télécharger et utiliser la "page web basique pour travailler".
  • Séparer le code Javascript dans un fichier séparé (même si c'est préférable en règle générale).
  • Noter l'instruction var (Chap. 3.3), puisqu'elle est maintenant obsolète.
  • Faire l'exercice 15.
  • Faire les activités du chapitre 7 (même si l'exercice 17 peut être un bon exercice d'approfondissement).

2. Que retenir ?

A minima...

  • Javascript est un langage de programmation qui permet de rendre une page web interactive.
  • Javascript est interprétable par les navigateurs web. Il permet donc une interactivité sur une page web, sans nécessité d'utiliser un serveur : c'est directement le navigateur qui interprète le script.
  • Pour utiliser Javascript, il faut simplement écrire le code entre deux balises <script></script>.
  • Tout comme le CSS et Brython, il est préférable de séparer le code Javascript dans un fichier séparé. Dans ce cas, il faut déclarer l'emplacement du fichier js dans la balise <head> du code HTML. Ex : <script src="mon_script.js"></script>
  • Avec Javascript, on peut écrire du code HTML dans une page, mais on peut aussi interagir avec ces éléments HTML, en changeant leur contenu, leur style, leur affichage,...

Au mieux...

  • Savoir ajouter des éléments HTML avec du code Javascript.
  • Savoir sélectionner des éléments HTML avec leur id ou leur class.
  • Savoir modifier un élément avec Javascript
  • Savoir coder une fonction pouvant s'éxecuter après tout événement sur la page (affichage, clic, menu déroulant, saisie d'un champ de texte, survol d'un élément,...).
